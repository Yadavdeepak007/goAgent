						<h1 class="entry-title">Find all  triplets with zero sum</h1>
				
						</header><!-- .entry-header -->

				<div class="entry-content">
			<p>Given an array of distinct elements. The task is to find triplets in array whose sum is zero.</p>
<p><strong>Examples :</strong></p>
<pre>
Input : arr[] = {0, -1, 2, -3, 1}
Output : 0 -1 1
         2 -3 1

Input : arr[] = {1, -2, 1, 0, 5}
Output : 1 -2  1
</pre>
<div id="practiceLinkDiv">
<h2><a href="https://practice.geeksforgeeks.org/problems/find-triplets-with-zero-sum/1">Recommended: Please solve it on &#8220;<b><i><u>PRACTICE</u></i></b>&#8221; first, before moving on to the solution.</a></h2>
</div>
<p><strong>Method 1 (Simple : O(n<sup>3</sup>))</strong><br />
The naive approach is that run three loops and check one by one that sum of three elements is zero or not if sum of three elements is zero then print elements other wise print not found.<br />

<p>&nbsp;</p>
<p><strong>Method 2 (Hashing : O(n<sup>2</sup>))</strong><br />
We iterate through every element. For every element arr[i], we find a pair with sum &#8220;-arr[i]&#8221;. This problem reduces to pairs sum and can be solved in O(n) time using hashing.</p>
<pre>
Run a loop from i=0 to n-2
  Create an empty hash table
  Run inner loop from j=i+1 to n-1
      If -(arr[i] + arr[j]) is present in hash table
         print arr[i], arr[j] and -(arr[i]+arr[j])
      Else
         Insert arr[j] in hash table.
</pre>

<p><strong>Time Complexity :</strong> O(n<sup>2</sup>)<br />

<p>&nbsp;</p>
<p><strong>Method 3 (Sorting : O(n<sup>2</sup>))</strong><br />
The above method requires extra space. We can solve in O(1) extra space. The idea is based on method 2 of <a href="https://www.geeksforgeeks.org/find-a-triplet-that-sum-to-a-given-value/">this</a> post.</p>
<pre>
1. Sort all element of array
2. Run loop from i=0 to n-2.
     Initialize two index variables l=i+1 and r=n-1
4. while (l &lt; r) 
     Check sum of arr[i], arr[l], arr[r] is
     zero or not if sum is zero then print the
     triplet and do l++ and r--.
5. If sum is less than zero then l++
6. If sum is greater than zero then r--
7. If not exist in array then print not found.
</pre>
